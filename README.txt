ASP.NET Core 7 MVC
Данный проект был реализован для просмотра, изменения, удаления данных о заказах в продуктовом магазине.
На сайт могут зайти только зарегестрированные пользователи. 
У каждой роли есть свой доступ к данным. 
Админ - управляет пользователями, ролями, может добавлять, изменять и удалять данные на сайте
Работник - имеет доступ к просмотру всех категорий, но не имеет право их изменять
